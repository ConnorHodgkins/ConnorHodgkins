Final Project

This is my final project for CSC 106. I decided to do a decision tree becuase it was one of my favorite assingments from this semester. My project is designed to help someone who is moving decide what state they would like to move to. It asks you questions like "Do you like warm weather?" which helps it eliminate states that dont match the description. After you have taken the quiz it gives you the option to look at more information about each state so you can be sure if you want to move there.
